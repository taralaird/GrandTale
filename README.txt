GrandTale
Submitted by: Tara Laird, Olivia Howard, Parsa Moeini 

Overview: 
This full version of GrandTale was based on the prototype version sent in by Tara Laird. 
This game includes the following features: 
- A Main Menu
- A How to Play Menu
- A Level Selction Screen that gets updated. Once the player finishes a level the next level is unlocked
- Three full playable levels including a movable and animated character with a staff weapon, animated enemies,
  scrolls to collect, UI that updates when a heart is lost or a scroll is collected, powerups (speed, 
  health, teleport), and the final level scrolls change position every time the level is replayed
- A Death/Lose Screen
- A Win Screen with a new letter for each level
- A intro story scene on the first level
- A final story scene after the final level is won, this scene also changes depending on if the player 
  killed any enemies/creatures
- A pause menu 
- And a soundtrack and sound effects

How to Execute:
To play the game, you will need to have Unity version 2022.3.9f1 installed. 
Open the build file and click on the GrandTale.exe file this will open a built version of the 
game which can be played. 

How to Look at Source Code/Design:
The game's Assets, Library, Packages, and ProjectSettings Unity folders have all been included. 
To view the game in Unity you can go to the Assets > Scenes and open the MainScene.unity file. 
All C# code used in the game can be seen in the Assets > Scripts folder. 

How to Play:
A easy explanation of how to play the game has been included in the executable game. See the 
How to Play button on the main menu to view the controls and main goal of the game.
